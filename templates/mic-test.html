HTTPS Setup Instructions

Option 1: Use ngrok (Recommended for Testing)

  1. Install ngrok: npm install -g ngrok
  2. Run: ngrok http 5000
  3. Use the HTTPS URL provided by ngrok

Option 2: Chrome Flag (Development Only)

  1. Open Chrome: chrome://flags/#unsafely-treat-insecure-origin-as-secure
  2. Enable the flag
  3. Add your IP: http://192.168.0.137:5000
  4. Relaunch Chrome

Option 3: Use localhost

Instead of accessing via IP (192.168.0.137), use: http://localhost:5000

Option 4: Add HTTPS to Flask (Production)


# Install Flask-SSL
pip install Flask-SSL

# In app.py:
from flask_ssl import SSL
ssl = SSL(app)
app.run(host='0.0.0.0', port=5000, ssl_context='adhoc')
